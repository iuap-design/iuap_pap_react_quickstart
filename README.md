
# iuap 前端开发框架（React）

## 一分钟快速上手

> 需要提前安装全局的 Node 环境

```
$ git clone git@github.com:yonyou-iuap/iuap_pap_react.git
$ cd iuap_pap_react
$ npm i     
# 三一集团开发使用以下命令安装依赖
$ npm run sanyinstall
$ npm run dev
```
